# Интерактивный тестер регулярных выражений
## Постановка задачи
Реализовать программу в виде оконного приложения, реализующего ввод регулярного выражения в строку ввода, его обработку, тестирование на входных данных и вывод на экран результата.
Учесть возможность неправильного ввода регулярного выражения.
Язык программирования - C++.
Разрешается использовать любые доступные фреймворки для построения оконных приложений на C++.
Запрещается использовать готовые библиотечные решения.
## Описание проекта 
Desktop-приложение на C++ с использованием фреймворка Qt демонстрирует работу регулярных выражений при поиске текстовых совпадений.
Пользователь может вводить текст и шаблон,загружать данные из файлов, сохранять тесты для дальнейшей работы, а также наблюдать подсветку всех найденных совпадений в реальном времени. Проект разрабатывался и тестировался на Qt 6.8.3 с компилятором MinGW 64-bit.
## Сборка и запуск проекта
### Требования
- Qt 6.8.х или новее (с компонентом MinGW или MSVC для Windows)

- Qt Creator (рекомендуется) или любой другой компилятор стандарта C++17 с поддержкой Qt:

- Git (для клонирования репозитория)
 
### Инструкция по сборке и запуску
#### Клонирование репозитория
Откройте терминал (командную строку) и выполните:

         git clone https://github.com/arichhhhka/Interact_tester.git // клонирование репозитория
         cd Interact_tester // переход в директорию проекта
#### Сборка проекта
#### Способ 1: Через Qt Creator (рекомендуется)
1) Запустите Qt Creator

2) Нажмите **Файл**, затем **Открыть файл или проект...**

3) Выберите файл **Kyrsovik.pro**

4) В появившемся окне выберите готовый комплект сборки (например, Desktop Qt 6.8.3 MinGW 64-bit)

5)  Нажмите **Configure Project**

6)  Соберите проект: нажмите на кнопку **"Собрать проект"** (молоток) или ипользуйте сочетание клавиш **Ctrl+B**  

7)  Запустите проект: нажмите **"Запустить"** (зеленый треугольник) или ипользуйте сочетание клавиш **Ctrl+R** 

####Способ 2: Из командной строки
 **Для Windows (с установленным Qt):**

1) Откройте **Qt Command Prompt**:
   - Нажмите **Пуск** → введите "Qt"
   - Выберите **Qt 6.8.3 (MinGW 64-bit)**
2) В открывшемся окне перейдите в папку с проектом:
   
            cd путь\к\проекту\Kyrsovik
**Примечание:**  Замените путь\к\проекту\Kyrsovik на путь, где находится ваш проект.

3) Создайте папку для сборки и перейдите в неё:

            mkdir build
            cd build

4) Запустите qmake:
   
            qmake ../Kyrsovik.pro

5) Соберите проект (для MinGW):
   
            mingw32-make

6) Готовый файл **Kyrsovik.exe** появится в папке **release**

**Для Linux**
1) Откройте терминал

2) Перейдите в папку с проектом:

           cd \путь\к\папке\Kyrsovik
   
3) Создайте папку для сборки и перейдите в неё:

           mkdir build
           cd build
   
4) Запустите qmake:

           qmake ../Kyrsovik.pro
   
5) Соберите проект:

           make
   
6) Запустите программу:

           ./Kyrsovik

**Для macOS**
1) Откройте Терминал (через Spotlight или Программы -> Утилиты)

2) Перейдите в папку с проектом:
  
           cd /путь/к/папке/Kyrsovik
   
3) Создайте папку для сборки и перейдите в неё:

           mkdir build
           cd build

4) Запустите qmake:

           qmake ../Kyrsovik.pro
   
5) Соберите проект:

           make
   
6) Запустите программу:

          open Kyrsovik.app
          или
          ./Kyrsovik
   
#### Способ 3: Cкачать готовую программу
Готовая сборка для Windows доступна в разделе [Releases](https://github.com/arichhhhka/Interact_tester/releases).

1. Скачайте **Kyrsovik_release.zip**
2. Распакуйте в любую папку
3. Запустите **Kyrsovik.exe**

**Примечание:** программа не требует установки Qt.
