Интерактивный тестер регулярных выражений
Постановка задачи
Реализовать программу в виде оконного приложения, реализующего ввод регулярного выражения в строку ввода, его обработку, тестирование на входных данных и вывод на экран результата. Учесть возможность неправильного ввода регулярного выражения. Язык программирования - C++. Разрешается использовать любые доступные фреймворки для построения оконных приложений на C++. Запрещается использовать готовые библиотечные решения.

Описание проекта
Desktop-приложение на C++ с использованием фреймворка Qt демонстрирует работу регулярных выражений при поиске текстовых совпадений. Пользователь может вводить текст и шаблон,загружать данные из файлов, сохранять тесты для дальнейшей работы, а также наблюдать подсветку всех найденных совпадений в реальном времени. Проект разрабатывался и тестировался на Qt 6.8.3 с компилятором MinGW 64-bit.

Сборка и запуск проекта
Требования
Qt 6.8.х или новее (с компонентом MinGW или MSVC для Windows)

Qt Creator (рекомендуется) или любой другой компилятор стандарта C++17 с поддержкой Qt:

Git (для клонирования репозитория)

Инструкция по сборке и запуску
Клонирование репозитория
Откройте терминал (командную строку) и выполните:

     git clone https://github.com/arichhhhka/Interact_tester.git // клонирование репозитория
     cd Interact_tester // переход в директорию проекта
Сборка проекта
Способ 1: Через Qt Creator (рекомендуется)
Запустите Qt Creator

Нажмите Файл, затем Открыть файл или проект...

Выберите файл Kyrsovik.pro

В появившемся окне выберите готовый комплект сборки (например, Desktop Qt 6.8.3 MinGW 64-bit)

Нажмите Configure Project

Соберите проект: нажмите на кнопку "Собрать проект" (молоток) или ипользуйте сочетание клавиш Ctrl+B

Запустите проект: нажмите "Запустить" (зеленый треугольник) или ипользуйте сочетание клавиш Ctrl+R

Способ 2: Из командной строки
Для Windows (с установленным Qt):

Откройте Qt Command Prompt:

Нажмите Пуск → введите "Qt"
Выберите Qt 6.8.3 (MinGW 64-bit)
В открывшемся окне перейдите в папку с проектом:

     cd путь\к\проекту\Kyrsovik
Примечание: Замените путь\к\проекту\Kyrsovik на путь, где находится ваш проект.

Создайте папку для сборки и перейдите в неё:

     mkdir build
     cd build
Запустите qmake:

     qmake ../Kyrsovik.pro
Соберите проект (для MinGW):

     mingw32-make
Готовый файл Kyrsovik.exe появится в папке release

Базовые символы и мета-символы
Символ	Описание	Пример	Совпадения
.	Любой одиночный символ	к.т	кот, кат, кит
\d	Любая цифра (0-9)	\d\d	12, 99, 05
\w	Словесный символ (буква, цифра)	\w+	hello, test123
\s	Пробельный символ (пробел, табуляция)	a\sb	a b
\D	Любой символ, кроме цифры	\D+	abc, hello
\W	Любой символ, кроме словесного	\W+	!@#, $
\S	Любой символ, кроме пробела	\S+	hello, 123
Квантификаторы (количество повторений)
Символ	Описание	Пример	Совпадения
*	0 или более повторений	ab*c	ac, abc, abbc
+	1 или более повторений	ab+c	abc, abbc (не "ac")
?	0 или 1 повторение	colou?r	color, colour
{n}	Ровно n раз	a{3}	aaa
{n,}	n или более раз	a{2,}	aa, aaa, aaaa
{n,m}	От n до m раз	a{2,4}	aa, aaa, aaaa
Якоря (позиции в строке)
Символ	Описание	Пример	Совпадения
^	Начало строки	^Hello	"Hello" в начале строки
$	Конец строки	end$	end в конце строки
\b	Граница слова	\bcat\b	cat (не "category")
\B	Не граница слова	\Bat\B	"at" внутри слова
Классы символов
Синтаксис	Описание	Пример	Совпадения
[abc]	Один из символов: a, b или c	[aeiou]	a, e, i, o, u
[^abc]	Любой, кроме a, b, c	[^0-9]	любые не-цифры
[a-z]	Диапазон символов	[a-z]	любая строчная буква
[A-Z]	Заглавные буквы	[A-Z]	любая заглавная буква
[0-9]	Цифры (аналог \d)	[0-9]	0, 1, 2
[a-zA-Z]	Любые буквы	[a-zA-Z]	a, Z, m
